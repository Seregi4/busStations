Для использования приложения пройдите по адресу:
1 http://localhost:8080/swagger-ui/index.html
2 Разверните метод гет
3 Нажмите Try it out
4 Введите first и second
5 Нажмите Execute

Снизу будет Code и  Response body

Можно также по url:
http://localhost:8080/api/direct/номер первой станции/номер второй станции

Текстовый файл с маршрутами можно поменять указав путь к вашему файлу в классе DirectServiceImpl в методе findDirect

В ресурсах в директории static лежат три тестовых файла
